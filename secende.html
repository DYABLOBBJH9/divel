تعلم CSS بأنيميشن بألوان متعددة

1. نط متكرر (Bounce)

هاد العنصر كيدير نط متواصل
@keyframes bounce {
  0%, 100% {transform: translateY(0);}
  50% {transform: translateY(-40px);}
}
.box {
  animation: bounce 3s ease-in-out infinite;
}

2. دوران مستمر (Spin)

هاد العنصر كيدور 360 درجة باستمرار
@keyframes spin {
  from {transform: rotate(0deg);}
  to {transform: rotate(360deg);}
}
.box {
  animation: spin 6s linear infinite;
}

3. نبض خفيف (Pulse)

هاد العنصر كيتنفس ببطء (تقل و يزيد الشفافية)
@keyframes pulse {
  0%, 100% {opacity: 1;}
  50% {opacity: 0.5;}
}
.box {
  animation: pulse 4s ease-in-out infinite;
}

4. انزلاق لليسار (Slide Left)

هاد العنصر كينزلق لليسار ويرجع
@keyframes slideLeft {
  0% {transform: translateX(0);}
  100% {transform: translateX(-70px);}
}
.box {
  animation: slideLeft 3s ease-in-out infinite alternate;
}

5. انزلاق لليمين (Slide Right)

هاد العنصر كينزلق لليمين ويرجع
@keyframes slideRight {
  0% {transform: translateX(0);}
  100% {transform: translateX(70px);}
}
.box {
  animation: slideRight 3s ease-in-out infinite alternate;
}

6. تغيير لون متكرر (Color Pulse)

هاد العنصر كيتغير لونو بين جوج ألوان
@keyframes colorPulse {
  0%, 100% {background-color: #e76f51;}
  50% {background-color: #f4a261;}
}
.box {
  animation: colorPulse 5s linear infinite;
}

7. هزة خفيفة (Shake)

هاد العنصر كيهتز يمين ويسار بسرعة
@keyframes shake {
  0%, 100% {transform: translateX(0);}
  25% {transform: translateX(-6px);}
  75% {transform: translateX(6px);}
}
.box {
  animation: shake 0.8s ease-in-out infinite;
}

8. قلب الدوران (Flip)

هاد العنصر كيدور على المحور Y
@keyframes flip {
  0%, 100% {transform: rotateY(0);}
  50% {transform: rotateY(180deg);}
}
.box {
  animation: flip 3s linear infinite;
}

9. تكبير وتصغير (Scale Up Down)

هاد العنصر كيتكبر وكينقص الحجم
@keyframes scaleUpDown {
  0%, 100% {transform: scale(1);}
  50% {transform: scale(1.3);}
}
.box {
  animation: scaleUpDown 4s ease-in-out infinite;
}

10. نبض الظل (Shadow Pulse)

هاد العنصر كينبض بالظل ديالو
@keyframes shadowPulse {
  0%, 100% {box-shadow: 0 0 15px;}
  50% {box-shadow: 0 0 40px;}
}
.box {
  animation: shadowPulse 3s ease-in-out infinite;
}
