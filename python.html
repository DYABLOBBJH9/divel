تعلم HTML مفصل وبالدارجة

شنو هي HTML؟

HTML هي لغة هيكلية كتسمح لنا نبنيو صفحات الويب. هي اللي كتحدد الهيكلة ديال المحتوى: نصوص، صور، فيديو، روابط، جداول، وغيرهم. بلا HTML، المتصفح مايعرفش كيفاش يعرض المحتوى ديالك. 


<!-- بداية صفحة HTML -->
<!DOCTYPE html>
<html>
  <head> ... </head>
  <body> ... </body>
</html>

هيكلة صفحة HTML

الصفحة كتبدأ بالتعريف <!DOCTYPE html> باش المتصفح يعرف نوع الصفحة.
من بعد كاين الوسوم <html>، <head>، و <body>.
<head> كتحتوي على معلومات ماكاتبانش ف الصفحة مباشرة، بحال العنوان والرابط لملفات CSS وجافاسكريبت.
<body> كتحتوي على المحتوى اللي غادي يبان ف الصفحة. 


<!DOCTYPE html>
<html>
  <head>
    <title>العنوان ديال الصفحة</title>
  </head>
  <body>
    المحتوى هنا
  </body>
</html>

الوسوم الأساسية (Tags)

HTML فيها بزاف ديال الوسوم اللي كتستعملها بزاف:
<h1> العنوان الكبير، <p> الفقرة، <a> الرابط، <img> الصورة، <ul> القائمة، وغيرها.
كل وسم كيبدأ بـ <tag> وكيخلص ب </tag> إلا كانت مش محتاجة تغلق. 


<h1>العنوان الكبير</h1>
<p>هادشي فقرة من النص.</p>
<a href="https://chat.openai.com">رابط للموقع</a>

الصفات (Attributes)

الوسوم كتقدر تكون عندها صفات، بحال href في الرابط، أو src في الصور.
الصفات كتزيد معلومات للوسم باش المتصفح يفهم كثر.
مثال: <a href="https://example.com">نص الرابط</a> 


<img src="img/photo.jpg" alt="صورة توضيحية">

كيفاش ندير صفحة بسيطة؟

دابا غادي نديروا مثال صغير على صفحة HTML كاملة وسهلة:


<!DOCTYPE html>
<html lang="ar" dir="rtl">
  <head>
    <meta charset="UTF-8">
    <title>صفحتي الأولى</title>
  </head>
  <body>
    <h1>مرحبا بكم!</h1>
    <p>هاد الصفحة كتوضح مثال بسيط.</p>
  </body>
</html>

نصائح باش تتعلم HTML مزيان

  • ديما جرب وكتب الكود بيدك، ماشي غير تقرا.
  • استعمل المتصفح باش تشوف النتيجة ديالك مباشرة.
  • تعلم تزيين الصفحات باستعمال CSS مع الوقت.
  • تابع المواقع التعليمية بحال MDN Web Docs بالعربية.

خلاصة طويلة وبسيطة

HTML هي العمود الفقري ديال أي صفحة ويب، بلاها مايمكنش تبني صفحة. كتعطي للمتصفح الهيكلة والمحتوى، والصفات كتزيد التفاصيل.
تعلمها سهل، ولكن خاصك التدريب والتجريب بزاف.
مورا HTML كتجي CSS باش تزين وتلون الصفحة، ومن بعد جافاسكريبت باش تزيد ديناميكية وحركة.
النصائح اللي عطيتك غادي تعاونك تبني صفحة جميلة ومفهومة.
واللي بغا يعرف أكثر، يقدر يشوف أكواد كبيرة ومتقدمة فالإنترنت ويتعلم منها.
