تعلم HTML بالدارجة المغربية - صفحة طويلة مع أنيميشن

1. شنو هي HTML؟

HTML هي اللغة اللي كتنظم بها صفحات الويب. كتعطي الهيكلة للمحتوى، بحال العناوين، الفقرات، الصور، والروابط.

<!-- مثال بسيط لصفحة HTML -->
<!DOCTYPE html>
<html>
  <head>
    <title>عنوان الصفحة</title>
  </head>
  <body>
    <h1>مرحبا بك!</h1>
  </body>
</html>

2. الوسم <h1> حتى <h6> - العناوين

كتستعمل العناوين باش تنظم الصفحة من الأكبر للأصغر. <h1> هو العنوان الكبير، <h6> صغير. 

<h1>العنوان الكبير</h1>
<h2>العنوان المتوسط</h2>
<h3>العنوان الصغير</h3>

3. الوسم <p> - الفقرة

الفقرة كتحتوي على نص، وكتكون دائما داخل وسم <p>.

<p>هذا نص داخل فقرة.</p>

4. الوسم <a> - الروابط

باش تدير رابط، كتستعمل وسم <a> مع خاصية href.

<a href="https://example.com">زر الرابط</a>

5. الوسم <img> - الصور

باش تحط صورة، كتستعمل وسم <img> مع الخاصيتين src و alt.

<img src="photo.jpg" alt="وصف الصورة">

6. الوسم <ul> و <ol> - القوائم

كتستعمل القوائم باش تنظم عناصر بشكل نقاط (غير مرتبة) أو أرقام (مرتبة).

قائمة غير مرتبة
<ul>
  <li>عنصر 1</li>
  <li>عنصر 2</li>
</ul>

قائمة مرتبة
<ol>
  <li>عنصر أ</li>
  <li>عنصر ب</li>
</ol>

7. الوسم <div> - الحاويات

وحدة من أهم الوسوم باش تجمع عناصر مع بعض وكتخدم معها CSS.

<div class="container">
  محتوى هنا
</div>

8. الوسم <span> - عنصر داخل سطر

باش تميز جزء صغير من النص داخل عنصر كبير.

<p>هنا <span style="color: red;">نص ملون</span>.</p>

9. الوسم <form> - النماذج

باش تجمع معلومات من المستخدم، كتستعمل <form>.

<form action="/submit" method="post">
  <input type="text" name="username" >
  <input type="submit" >
</form>

10. الوسم <table> - الجداول

كتستعمل الجداول باش تعرض بيانات منظمة في صفوف وأعمدة.

<table>
  <tr>
    <th>العنوان 1</th>
    <th>العنوان 2</th>
  </tr>
  <tr>
    <td>بيانات 1</td>
    <td>بيانات 2</td>
  </tr>
</table>

11. الوسم <header> - رأس الصفحة

كتستعمل باش تدير رأس الصفحة أو قسم علوي.

<header>
  <h1>عنوان الموقع</h1>
</header>

12. الوسم <footer> - تذييل الصفحة

باش تدير قسم التذييل في الأسفل.

<footer>
  حقوق النشر © 2025
</footer>

13. الوسم <nav> - التنقل

كتستعمل باش تدير قائمة التنقل داخل الموقع.

<nav>
  <a href="#home">الرئيسية</a>
  <a href="#about">معلومات</a>
</nav>

14. الوسم <section> - الأقسام

باش تجمع محتوى في قسم مستقل داخل الصفحة.

<section>
  محتوى القسم
</section>

15. الوسم <article> - مقال

كيستعمل لكتابة مقال مستقل أو محتوى قائم بذاته.

<article>
  مقال أو موضوع
</article>

16. الوسم <strong> و <b> - الخط العريض

باش تبرز نص بخط عريض.

<strong>نص مهم</strong>
<b>نص عريض</b>

17. الوسم <em> و <i> - الخط المائل

باش تبرز نص بخط مائل.

<em>نص مائل</em>
<i>نص مائل</i>

18. الوسم <br> - السطر الجديد

باش تدير سطر جديد بدون ما تبدأ فقرة جديدة.

<p>سطر هنا
</p>

19. الوسم <hr> - خط أفقي

باش تدير خط أفقي للفصل بين المحتويات.

<hr>

20. الوسم <meta> - ميتا البيانات

كتستعمل لتعريف بيانات الصفحة للمستعرض ومحركات البحث.

<meta charset="UTF-8">

21. الوسم <link> - الربط بالملفات الخارجية

باش تربط ملف CSS خارجي بالصفحة.

<link rel="stylesheet" href="style.css">

22. الوسم <script> - البرمجة

باش تضيف جافاسكريبت للصفحة.

<script>
  console.log('مرحبا بك في HTML!');
</script>

23. الوسم <video> - الفيديو

باش تضيف فيديو للصفحة.

<video controls>
  <source src="movie.mp4" type="video/mp4">
</video>

24. الوسم <audio> - الصوت

باش تضيف ملف صوتي.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

25. الوسم <iframe> - الإطار الداخلي

باش تضيف محتوى من موقع آخر داخل صفحتك.

<iframe src="https://example.com" width="600" height="400">
</iframe>
صفحة تعليم HTML طويلة ومفصلة بالدارجة المغربية مع أنيميشن وجمال في التنسيق.
